# node-blog

Projeto de blog responsivo desenvolvido com Node.js, Express, EJS e Sequelize.

## 🚀 Funcionalidades:
- Exibição de artigos na página inicial.
- Filtro de artigos por categoria.
- Crud de categorias e artigos.
- Layout responsivo.

## 🛠️ Tecnologias Utilizadas:
- **Node.js**
- **Express.js**
- **Sequelize (MySQL)**
- **EJS**
- **HTML5 e CSS3**
